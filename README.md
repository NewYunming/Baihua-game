# 白桦大冒险

一个基于 HTML5 Canvas 的像素风横版动作/肉鸽游戏，支持键鼠与触屏模式，无第三方运行依赖。

## 当前内容

- **武器体系**：剑、匕首、战斧、锤、巨剑、镰刀、长矛、双头刃、链刃，以及弓、手枪、步枪、冲锋枪、霰弹枪、手弩、卡宾枪、法杖和爆铳。新增武器均已接入品质池。
- **远程战斗**：半自动枪械基础伤害为原值的 50%，连发武器进一步降至 35% 并显著降低击退；弓保持 2 倍伤害、每箭固定装填 1 秒。霰弹枪的碎片增伤会在整次齐射内分摊。
- **永久碎片**：攻击、生命、近战范围、枪械弹容和装填速度在本局内永久叠加。攻击碎片会明确标注对远程武器按 66% 生效，切回近战后恢复完整效能；装填加速最高累计至 55%。
- **敌人能力**：行尸爪击、地精冲刺、兽人砸地、骷髅射箭、蜘蛛扑击、亡灵盾击、女妖激光和宝箱怪撕咬均有独立动作与像素造型。亡灵盾卫与幽魂女妖属于精英怪，每波各最多 1 个；盾卫不再回复护盾。怪物只有发现玩家或受到攻击后才会追击。
- **地图节奏**：玩家在地图中央出生，怪物按配额从左右两侧生成；玩家、普通怪和 Boss 均可跨越地图两端，并按环形最短路线追逐。怪物总数在所有模式中最多 20 只；之后生命、伤害与护盾采用缓慢的次线性成长，随机混沌怪已移除。
- **宝箱与白桦币**：普通地图至多生成 1 个宝箱，且只会放在底部连通、可步行或跳跃抵达的地面。攻击开启后的基础掉落为 1–5 枚白桦币，“开锁匠”可在此基础上追加奖励；部分宝箱会显露为宝箱怪。清场后会显示方向和距离指引，必须由玩家找到并处理宝箱后才进入结算或白桦商店。
- **武器奖励**：每次出现的三张武器卡最多可整组刷新 1 次，也可不选择新武器并保留当前装备；跳过武器不会跳过同波的永久碎片奖励。近战武器按品质提供额外生命上限。
- **白桦商店**：持有传说武器后，波次结算固定改为商店。主界面使用武器架、碎片簇和银/金/炫彩卡包图案；可花 2 枚白桦币任意更换传说武器、免费抽取一次碎片，或花 1/3/5 枚购买三档技能包。炫彩卡包与被动统一使用流动发光边框。
- **被动构筑**：最多 4 个槽位。现有 36 个不重复被动，三档各 12 个，覆盖破盾、远距校准、减速、击杀爆炸、反击、无伤任务、处决、引力、过量护盾、金币刃和环回冲击等机制。四个槽固定显示在画面底部并拥有独立像素 Logo；点击槽位或按 P 可暂停并查看全部详细描述、等级与任务进度。
- **主线与无尽**：第 10 波击败 Boss 后可保留当前成长进入无尽模式；每 10 波再次挑战 Boss。无尽模式死亡后会显示波次、击杀、精英、Boss、伤害、有效战斗时间、奖励选择、最终武器和成长数据；暂停及奖励页不会计入存活时间。
- **武器表现**：角色、怪物与武器统一为带暗色描边、材质高光和分层动作的写实像素画风；近战包含横斩、突刺、重劈、震地、蓄力横斩、镰割、旋刃和链舞等独立动画与有效命中窗口。
- **环境音乐**：草地、雪地、雨天、地狱、末地和星夜均有独立旋律、速度与音色；Boss 战使用专属战斗音乐。
- **画面反馈**：群系远景、天气、角色阴影、按模型对齐的枪口焰、弹道拖尾、盾击/破盾、伤害数字、激光预警、震屏及低动态偏好适配。

## 游玩方式
- 本地：双击 index.html 直接在浏览器打开。
- 线上（GitHub Pages）：推送到 GitHub 后，在仓库 Settings → Pages 启用，链接形如 `https://<你的用户名>.github.io/Baihua-game/`。

## 控制
- 键鼠：A/D 移动，空格跳跃，J/鼠标左键攻击，R 手动装填，U 求饶回血（约每秒+7HP），P 查看被动详情，Esc 打开或关闭暂停菜单。
- 暂停菜单：可继续游戏、重新开始和调节总音量；触屏模式还可调节操作按键大小。
- 触屏：选择“触屏模式”时会立即申请网页全屏；支持 Fullscreen API 的浏览器进入原生全屏，不支持任意网页元素全屏的浏览器则铺满可用视口。屏幕左侧方向键会独立抬高，默认尺寸只比主攻击键小一圈；右侧采用像素剑攻击键为中心的弧形布局，换弹键只在持有远程武器时出现。屏幕顶部中央为暂停键，操作区已禁用长按选择和复制菜单。
- 奖励：鼠标点击卡片，或按数字键 1/2/3；每次武器奖励按 F 最多全部刷新 1 次，按 S/0 不选择新武器；通关界面按 Enter/E 进入无尽模式，Esc 返回标题。
- 商店：主菜单可鼠标点击或按 1–6；武器/碎片/技能包用 1–3 选择，技能包按 F 整包刷新一次、S 放弃；满槽替换用 1–4 选择旧槽。
- 被动详情：点击底部任一被动槽或按 P 打开；此时游戏暂停，可用 1–4 切换高亮槽位，按 Esc 或点击关闭继续。
- 结算：死亡总结页按 Enter 重新开始主线，按 Esc 返回标题，也可点击对应按钮。

## 部署到 GitHub
1. `git init`
2. `git add .`
3. `git commit -m "Initial publish"`
4. 在 GitHub 创建公共仓库，例如 `Baihua-game`。
5. `git remote add origin https://github.com/<你的用户名>/Baihua-game.git`
6. `git branch -M main`
7. `git push -u origin main`
8. 仓库 Settings → Pages，Source 选 “Deploy from a branch”，Branch 选 main，Folder 选 /(root)，保存。
